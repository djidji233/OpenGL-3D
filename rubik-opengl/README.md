# Instructions

* **Arrows:** Move viewport angle
* **W / S:** Move Front / Back
* **P:** Show / Hide Walls
* **\+ / \-:** Increase / Decrease cube size
* **E:** Secret key
* **T / R:** Increase / Decrease cube transparency
* **B:** Show mirror reflection
